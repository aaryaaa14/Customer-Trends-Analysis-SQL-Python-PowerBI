# Customer Shopping Behavior Analysis

## Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI to uncover purchasing patterns, customer preferences, revenue trends, and business insights. The analysis helps businesses make data-driven decisions to improve customer engagement and sales performance.

## Objectives
- Analyze customer purchasing behavior
- Identify top-performing product categories
- Evaluate customer demographics and spending patterns
- Generate actionable business insights
- Create interactive dashboards for decision-making

## Tools & Technologies
- Python
- SQL
- Power BI
- Pandas
- NumPy
- Jupyter Notebook
- Data Visualization

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. SQL-Based Business Analysis
5. Customer Segmentation
6. Dashboard Development in Power BI
7. Insight Generation & Reporting

## Key Insights
- Customer purchasing trends across different demographics
- Revenue contribution by product categories
- Impact of customer loyalty and subscriptions
- Spending patterns and shopping preferences
- Regional and demographic sales performance

## Dashboard Features
- Revenue Analysis
- Customer Segmentation
- Product Performance Tracking
- Sales Trend Analysis
- Interactive Filters and KPIs

## Business Impact
The project demonstrates how data analytics can transform raw customer transaction data into meaningful insights that support strategic business decisions and improve overall performance.
